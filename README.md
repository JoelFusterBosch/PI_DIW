# PI_DIW
Este repositori conté el projecte PI_DIW, una aplicació web senzilla per a gestionar la reserva d’equipaments municipals. L’objectiu és permetre als usuaris consultar i reservar espais o serveis disponibles mitjançant pàgines estàtiques creades amb HTML, CSS i JavaScript.
## Què pots trobar en este projecte

El projecte inclou:

- Pàgines web estàtiques en HTML.
- Fulls d’estil CSS.
- Funcionalitats bàsiques amb JavaScript.
- Imatges de la interfície.
- Documentació i una presentació del projecte.

## Tecnologia utilitzada

- HTML per a l’estructura.
- CSS per als estils.
- JavaScript per a la interactivitat.
- Tot funciona de manera local, sense backend.

## Com utilitzar-lo
1. Clona el repositori

```bash
    git clone https://github.com/JoelFusterBosch/PI_DIW.git
```

2. Obri el projecte

- Accedeix a la carpeta.
- Obri el fitxer `.html` en el navegador.

## Estructura principal
- `reservesserveismunicipals.html`: Pàgina principal.
- `formulari_reserva.js`: Lògica del formulari.
- `css/`:  Estils del lloc web.
- `img/`: Imatges utilitzades.
- `Memòria.md`: Memòria del projecte.
- `Reserva d'equipaments municipals Projecte Intermodular.pptx`: Presentació.

### Notes
- És un projecte de caràcter educatiu.
- No disposa de base de dades ni servidor.
- Ideal per a practicar la creació d’interfícies i lògica bàsica amb JavaScript.
- Si surgeix una falla no dubteu en crear un `issues`.
